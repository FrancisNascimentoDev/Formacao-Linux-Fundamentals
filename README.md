# Formacao-Linux-Fundamentals
Aprenda a trabalhar com o principal sistema operacional usado em servidores de aplicações, desde a instalação até o gerenciamento de usuários com foco em segurança. Domine a manipulação de arquivos, permissões e os principais comandos Linux. Veja o poder do Linux na prática, com conteúdos essenciais para desenvolvedores e profissionais do mercado.
